Git Quiz

What is Git and why is it so important?

It's a distributed version control system that enables multiple developers to work on the codebase without breaking the application!

What's a branch and why do we create them?

A branch starts out as a copy of the main codebase. Any work done in the branch is isolated from the main branch so that it doesn't break the main application. It can be merged with the main application when it works.

What's the working tree?

The files in the initialised local repo that haven't been commited.

What's the staging area?

An area tha holds files waiting to be commited.

What's the command to retrieve the latest version of your files from the remote server?

git pull origin master (the first time), then just git pull.
